### Установка
1. Скачать репозиторий
2. `yarn install`
3. `gulp`

### Код-гайд
_С большинством настроек поможет включённый в шаблон .editorconfig_
1. Используем двойные кавычки везде, кроме `js`-файлов. В них одинарные

### Соглашение по неймингу изображений
